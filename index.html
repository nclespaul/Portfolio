NY Times Article Search

This React.js app uses the NY Times Article Search API to comb the Times for articles of interest to you. You can use keywords to search for articles spanning any time period, and the search will return clickable links to take you to anything that catches your interest. You can also save the articles to the database for later, if you wish.

Tools Used: HTML5, CSS, Bootstrap, Javascript, Node.js, React.js, MongoDB, Express, Heroku, NY Times API, NPM's (axios, body-parser, express, mongoose, morgan, react, react-dom, react-router), Dev Dependencies (babel-core, babel-loader, babel-preset-es2015, babel-preset-react, webpack)

The M.A.R.S. Project

Math anxiety affects approximately 20% of the population to varying degrees, and is quantified using a standardized survey. This app is intended for use as a tool for filtering the results of that survey to provide information to researchers and administrators for use in tailoring budgets, resources, and techniques towards reducing the anxieties experienced by the particular subgroup they are interested in.

Tools Used: HTML5, CSS3, Javascript, JQuery, JSON, Plotly.js, Node.js, Express, Sequelize, MySQL, Github, Heroku, NPM's (express, body-parser, mysql, mysql2, path, sequelize, sequelize-cli)

Full Stack Warrior

The Full Stack Warrior app is a Skyrim themed trivia game consisting of eight questions that must be answered within sixty seconds. The questions do not appear until the start button is pressed, starting the timer. Once the time expires, the number of correct and incorrect questions is displayed. The game can then be started over if desired.

Tools Used: HTML, CSS, Bootstrap, Javascript, JQuery.

Train Scheduler

All Aboard!! This simple app allows entry of incoming train information, including the train name, the destination, and the frequency of arrival. The display is then updated with the arrival time of the next incoming train, as well as the number of minutes until arrival.

Tools Used: HTML5, CSS3, Javascript, Bootstrap, Jquery, Moment.js, Firebase, Github Pages.
